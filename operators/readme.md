## Categories & Symbols

- Arithmetic  
    - Addition: `+`  
    - Subtraction: `-`  
    - Multiplication: `*`  
    - Division: `/`  
    - Remainder: `%`  
    - Exponentiation: `**`  
    - Increment / Decrement: `++`, `--`

- Assignment  
    - Simple: `=`  
    - Compound: `+=`, `-=`, `*=`, `/=`, `%=`, `**=`, `<<=`, `>>=`, `>>>=`, `&=`, `|=`, `^=`

- Comparison  
    - Equality: `==`, `!=`  
    - Strict equality: `===`, `!==`  
    - Relational: `>`, `<`, `>=`, `<=`

- Logical  
    - AND: `&&`  
    - OR: `||`  
    - NOT: `!`

- Bitwise  
    - AND `&`, OR `|`, XOR `^`, NOT `~`  
    - Shifts: `<<`, `>>`, `>>>`

- Unary / Miscellaneous  
    - `typeof`, `void`, `delete`  
    - Unary plus/minus: `+x`, `-x`  
    - Comma operator: `,`

- Ternary  
    - Conditional: `condition ? exprIfTrue : exprIfFalse`

- Modern ES operators  
    - Nullish coalescing: `??`  
    - Optional chaining: `?.`  
    - Spread / Rest: `...`

- Member & Call  
    - Property access: `.`, `[]`  
    - Function call: `()`

## Quick examples
- Arithmetic: `a = b + 2 * c;`  
- Strict compare: `if (a === b) { ... }`  
- Nullish: `const x = maybeNull ?? defaultVal;`  
- Optional chaining: `user?.address?.city`  
- Spread/rest: `fn(...args)` / `const [a, ...rest] = arr`
